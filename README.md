# usta-norcal-ratings

Puropose: I decided to create a USTA NorCal database on all the male players who played the 18+ season last year in 2025. This is a for fun project since I am an avid tennis player also. 

The project has an SQL analysis of the USTA NorCal player ratings dataset that I created on PostgreSQL, and a PowerBI Dashboard of USTANorcal Players.
